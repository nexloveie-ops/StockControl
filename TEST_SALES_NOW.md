# 🛒 立即测试：新销售业务

## 快速开始（3分钟）

### 准备工作
1. 访问 http://localhost:3000
2. 登录商户账号：`merchant_001` / `merchant123`
3. 确保有库存（如果没有，先去"我的库存"手动入库）

### 测试 1：浏览分类
1. 默认进入"销售业务"标签
2. **看到什么**：彩色的分类卡片
3. **显示内容**：分类名称、产品种类数、库存数

### 测试 2：查看产品
1. 点击任意分类卡片
2. **看到什么**：该分类下的所有产品列表
3. **每个产品显示**：
   - 产品名称（大字）
   - 品牌、型号、颜色
   - 库存数量（蓝色标签）
   - 价格（绿色大字）
   - "加入购物车"按钮（绿色）

### 测试 3：添加到购物车
1. 点击任意产品的"+ 加入购物车"按钮
2. **看到什么**：
   - 弹出提示"✅ [产品名] 已添加到购物车"
   - 页面顶部出现蓝色购物车摘要
   - 显示"🛒 购物车：1 件商品 总计：€X.XX"

### 测试 4：添加多个产品
1. 继续点击其他产品的"加入购物车"
2. 或者多次点击同一产品（数量+1）
3. **看到什么**：购物车数量和金额实时更新

### 测试 5：现金支付
1. 点击购物车摘要中的"💳 结账"按钮
2. **看到什么**：结账模态框
   - 购物车明细
   - 客户电话输入框
   - 三个支付方式按钮
3. 点击"💵 现金"按钮（按钮会高亮）
4. 点击"✅ 确认支付"
5. **看到什么**：
   - "✅ 销售成功！"消息
   - 显示订单号和金额
   - 购物车清空
   - 返回分类列表

### 测试 6：混合支付
1. 添加产品到购物车（假设总额€10）
2. 点击"💳 结账"
3. 点击"💰 混合"按钮
4. **看到什么**：
   - 显示混合支付输入框（蓝色背景）
   - 自动填充现金€5.00，刷卡€5.00
5. 修改金额测试：
   - 现金改为€3
   - 刷卡改为€5
   - **看到什么**：红色警告"支付总额 €8.00 与订单总额 €10.00 不符"
6. 修正金额：
   - 现金改为€6
   - 刷卡改为€4
   - **看到什么**：警告消失
7. 点击"✅ 确认支付"
8. **看到什么**：销售成功

### 测试 7：验证库存
1. 完成销售后，点击"我的库存"标签
2. 找到刚才销售的产品
3. **看到什么**：库存数量已减少

## 常见问题

**Q: 没有看到分类卡片？**
A: 确保库存中有产品，且数量 > 0。

**Q: 购物车摘要不显示？**
A: 购物车为空时不显示，添加产品后会自动出现。

**Q: 混合支付一直提示金额不匹配？**
A: 确保现金金额 + 刷卡金额 = 订单总额（精确到分）。

**Q: 点击结账没反应？**
A: 检查浏览器控制台是否有错误，或刷新页面重试。
