# 🚀 用户管理和群组管理 - 快速参考卡

## 📍 访问地址
```
http://localhost:3000/admin.html
```

---

## 🎯 核心功能

### 用户管理
| 功能 | 操作 | 位置 |
|------|------|------|
| 查看用户 | 点击"👥 用户管理"标签 | 主界面 |
| 添加用户 | 点击"➕ 添加用户"按钮 | 用户管理页面 |
| 编辑用户 | 点击用户行的"编辑"按钮 | 用户列表 |
| 停用用户 | 点击用户行的"停用"按钮 | 用户列表 |
| 筛选用户 | 使用角色/状态/群组筛选器 | 用户管理页面 |
| 搜索用户 | 在搜索框输入用户名 | 用户管理页面 |

### 群组管理
| 功能 | 操作 | 位置 |
|------|------|------|
| 打开群组管理 | 点击"👥 群组管理"按钮 | 用户管理页面 |
| 添加群组 | 点击"➕ 添加群组"按钮 | 群组管理模态框 |
| 编辑群组 | 点击群组行的"编辑"按钮 | 群组列表 |
| 停用群组 | 点击群组行的"停用"按钮 | 群组列表 |
| 查看用户数 | 查看"用户"列 | 群组列表 |

## 🎨 按钮和颜色

### 按钮
- 🔴 **红色按钮** - 主要操作（添加用户、添加群组、保存）
- 🟢 **绿色按钮** - 次要操作（群组管理）
- ⚪ **灰色按钮** - 取消操作

### 徽章
- 🔴 **管理员** - 红色
- 🟡 **仓库管理员** - 黄色
- 🔵 **普通用户** - 蓝色
- 🟢 **活跃** - 绿色
- 🔴 **停用** - 红色
- 🔵 **群组代码** - 蓝色

## 📝 必填字段

### 创建用户
- ✅ 用户名（唯一）
- ✅ 邮箱（唯一）
- ✅ 密码（至少6位）
- ✅ 角色

### 创建群组
- ✅ 群组名称（唯一）
- ✅ 群组代码（唯一，自动大写）

## ⚡ 快速测试流程

### 1分钟测试
```
1. 访问 http://localhost:3000/admin.html
2. 点击 "👥 用户管理"
3. 点击 "👥 群组管理" → "➕ 添加群组"
4. 填写: 名称=测试群组, 代码=TEST
5. 保存 → 关闭
6. 点击 "➕ 添加用户"
7. 填写: 用户名=test, 邮箱=test@test.com, 密码=123456, 角色=普通用户
8. 选择群组=测试群组
9. 保存
10. 验证用户列表显示新用户和群组
```

## ⚠️ 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 用户名已存在 | 用户名重复 | 使用不同的用户名 |
| 邮箱已存在 | 邮箱重复 | 使用不同的邮箱 |
| 无法删除群组 | 群组有用户 | 先移除或停用用户 |
| 点击无反应 | JS错误 | 打开控制台查看错误 |
| 保存失败 | API错误 | 查看Network标签 |
