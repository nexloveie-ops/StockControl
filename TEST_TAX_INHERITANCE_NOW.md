# 🧪 立即测试：税率自动继承

## 快速测试（5分钟）

### 步骤 1：配置分类税率
1. 访问 http://localhost:3000
2. 登录管理员账号：`admin` / `admin123`
3. 进入"系统设置"
4. 在"产品分类管理"中添加或编辑分类：
   - 分类类型：`Type-C Cable 1M`
   - 默认税率：`VAT 23%`
   - 点击"保存"

### 步骤 2：商户入库
1. 退出管理员账号
2. 登录商户账号：`merchant_001` / `merchant123`
3. 点击"我的库存"标签
4. 点击"+ 手动入库"按钮
5. 填写信息：
   ```
   产品名称：Type-C Cable 1M
   分类：Type-C Cable 1M（从下拉框选择）
   数量：10
   成本价：2.00
   批发价：3.00
   零售价：5.00
   ```
6. 点击"确认入库"

### 步骤 3：验证结果
在"我的库存"列表中查看：
- ✅ 产品名称：Type-C Cable 1M
- ✅ 分类：显示分类标签
- ✅ **税务分类：应显示"VAT 23%"蓝色徽章** ⭐

## 预期结果
税务分类列应该自动显示 `VAT 23%`，无需手动选择！
